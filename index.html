Générateur de Script pour Déploiement GitHub Pages

Cet outil génère un script à copier-coller dans votre terminal pour publier un nouveau projet sur GitHub Pages. Remplissez les champs et suivez attentivement les instructions.

Partie 1 : Création du Dépôt sur GitHub.com

  1. Allez sur https://github.com/new pour créer un nouveau dépôt.
  2. Repository name : Entrez "". N'utilisez pas d'espaces dans ce nom. Utilisez des tirets ou underscores.
  3. Assurez-vous que "Public" est sélectionné.
  4. NE COCHEZ AUCUNE CASE d'initialisation ("Add a README file", "Add .gitignore", "Choose a license"). Le dépôt doit être créé complètement vide.
  5. Cliquez sur "Create repository".
  6. Sur la page suivante (celle avec "Quick setup... or create a new repository on the command line"), repérez et copiez l'URL HTTPS. Elle ressemblera à :
    https://github.com//.git
    Le script terminal ci-dessous utilisera cette URL (elle y sera déjà insérée).

Partie 2 : Script à Copier-Coller dans votre Terminal

Ouvrez votre Terminal (ou Git Bash sur Windows). Assurez-vous d'être dans votre dossier "home" ou un dossier parent de votre projet (pas à l'intérieur du dossier du projet lui-même) avant de coller, car la première commande est cd "CHEMIN_DU_PROJET".

Copiez TOUT le bloc de commandes ci-dessous et collez-le en une seule fois dans votre terminal. Appuyez sur Entrée. Le script s'exécutera étape par étape.
